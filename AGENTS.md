# AI Agent Rules

이 프로젝트는 Astro 기반 개인 홈페이지입니다.

## 프로젝트 목표

- 최종 배포 목표는 GitHub Pages의 `github.io` 주소입니다.

## 작업 규칙

- 새 라이브러리는 사용자 승인 없이 추가하지 않습니다.
- 관련 없는 파일은 수정하지 않습니다.
- `secret`, `token`, `password`, `API key` 등 민감한 정보는 코드에 작성하지 않습니다.
- 삭제 명령은 실행 전에 반드시 사용자에게 확인합니다.
- 수정 후에는 변경한 파일과 그 이유를 설명합니다.
- 가능하면 변경 후 `npm run build`로 확인합니다.
- 변경사항이 생기면 자주 commit합니다.

## Commit 규칙

커밋 메시지는 아래 형식을 사용합니다.

- `feat: 한국어설명`
- `fix: 한국어설명`
- `chore: 한국어설명`
- `docs: 한국어설명`
